# Active Context

## Current Focus

No active work in progress. This file tracks in-flight tasks and decisions being made *right now*. When a task completes, its outcome is folded into the permanent architecture reference (`docs/`) and the feature summary (`progress.md`) — this file is then cleared back to this template so the next session starts with a clean slate.

## How to Use This File

- While a task is in progress: record the problem being solved, the decision being made, and any open questions here.
- Once the task is done and verified: move durable facts into `docs/*.md` (architecture/behavior) or `progress.md` (feature inventory), then reset this file to "No active work in progress."
- Do **not** accumulate a chronological changelog here — that's what version control is for. This file answers "what am I doing right now," not "what did we do last month."
